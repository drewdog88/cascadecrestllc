# Oregon / Portland rental screening compliance

**Not legal advice.** Use with ORHA, PAROA, or counsel.

## What this site does (minimal PII)

| Requirement | Where |
| --- | --- |
| Written screening criteria before screening fee | [/screening-criteria](/screening-criteria) — applicant reads before RentSpree |
| ORS 90.295 disclosures | [/apply](/apply) + screening criteria |
| Uniform criteria | Same published criteria + same RentSpree flow for everyone |
| Fair Chance notice + acknowledgment | [/apply](/apply) checkboxes (not stored server-side) |
| FCRA-compliant screening | RentSpree / TransUnion only |
| Fee receipt | RentSpree (not this site) |
| First-come, first-served (Portland) | Use RentSpree application timestamps |
| Landlord checklist | [/compliance](/compliance) |

## What this site does **not** do

- Store names, addresses, income, rental history, or ID
- Send application data through our API or database
- Email applicant PII from our servers

## Your operational responsibilities

1. Publish real thresholds (credit/income/rent) in screening criteria when listing a unit.
2. Process complete applications in order using **RentSpree** timestamps (Portland).
3. Document individualized criminal-history assessments and adverse action in RentSpree / your files.
4. Retain partner receipts and reports—not on this website.

## PII posture

**cascadecrestllc.com:** disclosures only. **RentSpree:** system of record for applications and screening.
